# 📊 Web Data Monitoring & QA System

A multi-page web scraping and data monitoring system built using Python, Pandas, and Streamlit.

## 🚀 Features

- Multi-page web scraping
- Data cleaning & validation
- Anomaly detection using Z-score
- Interactive dashboard (Streamlit)
- KPI metrics & visualization

## 🛠 Tech Stack

- Python
- Pandas
- BeautifulSoup
- Streamlit
- Matplotlib

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run scraper:
python scraper.py

3. Run dashboard:
streamlit run dashboard.py
